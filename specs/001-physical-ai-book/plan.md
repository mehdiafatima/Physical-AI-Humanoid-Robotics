This plan outlines the generation of a complete, production-ready Docusaurus book based on the `/sp.constitution` specification. The plan will involve cleaning the existing `docs` directory, creating a new directory structure with numeric prefixes for ordering, and using `_category_.json` for clean sidebar labels.
